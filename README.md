# 🏝️ Сбежать с острова Эштейна (Escape from Eshtein Island)

## 📖 Описание проекта
Проект представляет собой прототип мобильной игры в жанре Survival/Adventure на движке Unity. 
Игрок управляет персонажем от первого лица, исследует остров, взаимодействует с объектами окружения (открытие ящиков, сбор лута) и выживает в условиях смены дня и ночи.

### 🔑 Ключевые механики:
- **Система времени**: Динамическая смена дня и ночи (цикл 24 часа) с изменением освещения, цвета неба (Skybox) и интенсивности источников света.
- **Взаимодействие**: Raycast-система (SphereCast) для взаимодействия с объектами на расстоянии (открытие/закрытие контейнеров), также взаимодействие через OnTriggerEnter.
- **ИИ врагов**: NPC с навигацией по NavMesh, реализующие преследование игрока с оптимизацией пересчета пути.
- **Атмосфера**: Реалистичная смена освещения через один Directional Light с использованием цветовой температуры (Color Temperature).


## 🛠 Зависимости (Dependencies)
Проект разработан на движке **Unity**. Для сборки и запуска необходимы следующие компоненты:

1. **Unity Hub & Unity Editor**: Рекомендуемая версия `6000.3.x LTS` или новее.
2. **Пакеты Unity (через Package Manager)**:
   - `com.unity.modules.ai` (NavMesh Agent для ИИ)
   - `com.unity.render-pipelines.universal` (URP - продвинутое освещение и скайбоксы)
   - `com.unity.textmeshpro` (Отображение UI текста)
3. **.NET Framework**: Версия `.NET Standard 2.1` (встроена в Unity).

> *Примечание: Проект не использует внешние менеджеры сборки типа Gradle/Maven, так как компиляция и линковка осуществляются встроенными средствами Unity Engine.*

## 🚀 Как запустить проект

### Способ 1: Запуск готовой сборки (Рекомендуемый)
Проект уже скомпилирован и готов к запуску.
1. Перейдите в папку проекта: `/Builds` (или `/Bin`).
2. Найдите исполняемый файл: **`EscapeFromEinstein.exe`** (название может отличаться в зависимости от настроек сборки).
3. Дважды кликните по файлу для запуска игры.
   - *Требования*: ОС Windows 10/11, видеокарта с поддержкой DirectX 11.

### Способ 2: Запуск через Unity Editor (Для разработчиков)
1. Клонируйте репозиторий:
   ```bash
   git clone <ссылка_на_твой_репозиторий>
2. Откройте Unity Hub, нажмите Add и выберите папку с проектом.
3. Дождитесь импорта ассетов и компиляции скриптов.
4. Нажмите кнопку Play в редакторе для тестирования.
