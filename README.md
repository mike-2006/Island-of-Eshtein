# 🏝️ Сбежать с острова Эштейна (Escape from Einstein Island)

## 📖 Описание проекта
Проект представляет собой прототип мобильной игры в жанре Survival/Adventure на движке Unity. 
Игрок управляет персонажем от первого лица, исследует остров, взаимодействует с объектами окружения (открытие ящиков, сбор лута) и выживает в условиях смены дня и ночи.

### 🔑 Ключевые механики:
- **Система времени**: Динамическая смена дня и ночи с изменением освещения и скайбоксов.
- **Взаимодействие**: Raycast-система для взаимодействия с объектами (открытие контейнеров).
- **ИИ врагов**: NPC с навигацией по NavMesh, преследующие игрока.
- **Инвентарь**: Базовая система подбора предметов.

## 🛠 Зависимости (Dependencies)
Проект разработан на движке **Unity**. Для корректного запуска необходимы следующие компоненты:

1. **Unity Hub & Unity Editor**: Версия `2022.3.x LTS` (или новее).
2. **Пакеты Unity (через Package Manager)**:
   - `com.unity.modules.ai` (NavMesh Agent)
   - `com.unity.render-pipelines.universal` (URP - освещение)
   - `com.unity.textmeshpro` (UI текст)
   - `com.unity.visualscripting` (опционально)
3. **.NET Framework**: Версия `.NET Standard 2.1` (встроена в Unity).

> *Примечание: Проект не использует внешние менеджеры сборки типа Gradle/Maven, так как сборка осуществляется средствами Unity Engine.*
